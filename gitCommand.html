1.git init ->: this command is used to give access to git for repo

2.git remote add origin ..remote_repo_URL->: this command is used to create/add local repo same as remote repo

3.git push origin master>

4.git pull --rebase origin main ->: this command update the local repo with origin repo(if remote repo is commit ahead than local )

5.git pull

6.git pull --merge

7.git add .->: this add all the file inside the repo to our local branch

8.git commit -m "this will save the chages in added file to our local repo"

9.git reset ->: this will unstage the file from staging area to workspace

10.git push -f origin 'branch name Remote' -> this help us to push file forcefully to remote repo

11. git reset HEAD -- . -> this command will help us to remove file from stage area to working directory

12. git clone -b {branch_name} {repo clone link}

13.after cloning change to directory and run npm i {this command will install all the files that are required to run the app}
